# Jhelum Foundation Website

Official website for Jhelum Foundation - Restoring rivers, restoring lives.

## Project Structure

```
jhelum-foundation/
├── index.html              # Main HTML file
├── css/
│   ├── style.css          # Main stylesheet
│   └── responsive.css     # Responsive design styles
├── js/
│   ├── main.js            # Main JavaScript functions
│   └── animations.js      # Animation effects
├── images/                # Image assets folder
│   ├── hero-bg.jpg
│   ├── about-image.jpg
│   └── programs/
└── README.md              # This file
```

## Setup Instructions

1. **Download all files** and maintain the folder structure as shown above.

2. **Replace placeholder images** in the `images/` folder:
   - `hero-bg.jpg` - Background for hero section
   - `about-image.jpg` - Image for about section
   - Program images in `programs/` folder

3. **Update contact information** in `index.html`:
   - Email address
   - Phone number
   - Physical address
   - Social media links

4. **Customize content** as needed in `index.html`.

5. **Open** `index.html` in a web browser to view the site.

## Customization

### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-blue: #1e3a8a;
    --secondary-blue: #3b82f6;
    --accent-green: #10b981;
    /* ... */
}
```

### Fonts
Change the font family in `css/style.css`:
```css
body {
    font-family: 'Your Font Here', sans-serif;
}
```

## Image Guidelines

### Recommended Sizes:
- **Hero Background**: 1920x1080px
- **About Image**: 800x600px
- **Program Cards**: 800x600px

### Format:
- JPG for photographs
- PNG for images with transparency
- Optimize images before upload (use tools like TinyPNG)
